# Examples

Add public, non-sensitive demonstrations here.

Recommended format for each example:

```text
examples/<example-name>/
├── reference.png
├── recreated.drawio
├── recreated.png
├── SVG_ASSETS.md
└── svg/
```

Good launch examples:

- academic model figure
- cloud architecture screenshot
- business workflow diagram

Do not commit private customer diagrams, screenshots with sensitive text, or protected brand assets unless they are explicitly cleared for public release.
